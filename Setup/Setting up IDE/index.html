    Choosing the IDE or Code Editor

    I mostly prefer Visual Studio Codium for easy development and fast setup
    You can use PyCharm Community Edition as most proffessional developers use it but the con of this it works kinda slow on mid-low end computers/laptops and downloading the installor is mostly big
    You might say why not Visual Studio Code?

    VSCode vs VSCodium

    Visual Studio is Open-Source-Software but the binaries are not builded from Open-Source Code
    While Visual Studio Codium is a manual builded version with patches for telemetry being disabled but also there is no some of extensions from VSCode as it uses Open VSX

    For Linux

    VSCodium

    For Arch-based distros:

    sudo pacman -S vscodium

    For Other distros please download it from VSCodium WebSite
    After downloading appropriate file then install it with these commands For Debian-based distros:

    sudo apt install <file.deb>

    For RHEL-based distros:

    sudo yum install <file.rpm>
or
sudo dnf install <file.rpm>

    For Fedora-based distros:

    sudo dnf install <file.rpm>

    For Gentoo-based distros: Sorry, dig it out for yourself as you are pro
